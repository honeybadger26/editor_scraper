# editor_scraper

## Instructions

1. Requires **Python 3.9**
2. Install `requests` and `beautifulsoup4` for Python
```
pip install -r requirements.txt
```
Alternatively use Python virtual environment:
```
python -m venv env
source env/bin/activate
pip install -r requirements.txt
```
3. Run
```
python app.py
```

## Notes

Run tests
```
python -m unittest tests -b
```
